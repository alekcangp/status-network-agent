## Session 11: Intent Detection Refinement
Date: 2026-03-19

### Issue:
Initial LLM prompt was misclassifying messages. "store number 5224" was stored but "get all notes" was also being stored instead of retrieved. The AI was treating everything as STORE.

### Solution:
Updated prompt to be clearer:
- STORE: if user wants to save/remember/store any information
- RETRIEVE: if user wants to see stored notes  
- CHAT: ONLY for pure casual conversation (greetings, small talk)

### Testing Results:
- "remember 123" → STORE ✅
- "show my notes" → RETRIEVE ✅
- "hi" → CHAT ✅
